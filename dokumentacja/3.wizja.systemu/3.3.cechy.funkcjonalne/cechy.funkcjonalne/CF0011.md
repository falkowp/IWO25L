#### (CF0011) Automatyzacja rejestracji firm

|               |           |
|---------------|-----------|
| **Priorytet** | _wysoki_  |
| **Trudność**  | _średnia_ |

##### Opis

System powinien
umożliwiać pracownikom jednostki samorządowej automatyzację procesu rejestracji firm, aby skrócić czas obsługi i zwiększyć efektywność.
