#### (CJF0004) Zgodność systemu z normami technologicznymi

|               |                  |
| ------------: |-----------------:|
|       **Typ** | *funkcjonalność* |
| **Priorytet** |              *A* |
|  **Trudność** |        *średnia* |


##### Opis
System powinien gwarantować zgodność z normami technologicznymi. 

##### Sposób pomiaru
Wynajęcie osoby kompetentnej do wykonania oceny czy system jest zgodny z normami.
