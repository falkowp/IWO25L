#### (CJL0002) Łatwy w modyfikacji

|               |                      |
| ------------: |---------------------:|
|       **Typ** | *łatwość utrzymania* |
| **Priorytet** |                  *B* |
|  **Trudność** |            *średnia* |


##### Opis
System musi być łatwy w modyfikacji

##### Sposób pomiaru
Czas jaki zajmuje dodanie nowej funkcjonalności po roku

##### Możliwe wyniki pomiaru

1 godzina do 1 roku

##### Akceptowalne wartości pomiaru

do 2 tygodni
