#### (CJF0003) Zgodność danych z prawem

|               |                  |
| ------------: |-----------------:|
|       **Typ** | *funkcjonalność* |
| **Priorytet** |              *A* |
|  **Trudność** |        *średnia* |


##### Opis
System musi gwarantować zgodność z prawem. 

##### Sposób pomiaru
Wynajęcie osoby kompetentnej do wykonania audytu prawnego. Ilość elementów niezgodnych z prawem.

##### Możliwe wyniki pomiaru
od 0 do 1000

##### Akceptowalne wartości pomiaru
0
