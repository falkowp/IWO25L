#### (CJW0001) System powinien być skalowalny

|               |             |
| ------------: |------------:|
|       **Typ** | *wydajność* |
| **Priorytet** |         *A* |
|  **Trudność** |   *średnia* |


##### Opis
System powinien być skalowalny.

##### Sposób pomiaru
Zwiększenie zasobów systemu zezwala na obsłużenie większej liczby użytkowników

##### Możliwe wyniki pomiaru
Tak / Nie

##### Akceptowalne wartości pomiaru
Tak
