### (CF0003) Rejestracja i aktualizacja informacji o źródłach danych

|               |            |
|---------------|------------|
| **Priorytet** | _kluczowy_ |
| **Trudność**  | _średnia_  |

##### Opis

System musi
umożliwiać [pracownikom jednostki samorządowej](../../3.2.interesariusze/interesariusze/pracownik_jednostki_samorzadowej.md)
rejestrację i aktualizację informacji o źródłach danych, aby zapewnić aktualność i wiarygodność udostępnianych
informacji.
