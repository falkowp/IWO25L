#### (CF0006) Wizualizacja danych

|               |           |
|---------------|-----------|
| **Priorytet** | _niski_   |
| **Trudność**  | _średnia_ |

##### Opis

System może umożliwiać mieszkańcom przyjazną wizualizację danych, w tym geoprzestrzenną, aby ułatwić interpretację i wykorzystanie informacji.
