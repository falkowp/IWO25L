### (CF0007) Możliwość oceny danych

|               |           |
|---------------|-----------|
| **Priorytet** | _wysoki_  |
| **Trudność**  | _średnia_ |

##### Opis

System powinien umożliwiać [mieszkańcom](../../3.2.interesariusze/interesariusze/mieszkaniec.md) ocenę jakości danych,
aby zwiększyć ich wiarygodność i przydatność.
