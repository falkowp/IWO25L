#### (CF0001) Agregacja danych

|               |            |
|---------------|------------|
| **Priorytet** | _kluczowy_ |
| **Trudność**  | _średnia_  |

##### Opis

System musi umożliwiać administratorowi systemu łączenie danych z różnych źródeł w jednym miejscu, aby zapewnić spójny i łatwy dostęp do informacji.
